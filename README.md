# Kotlin Laboratory Exercises

This repository is dedicated to laboratory exercises using the **Kotlin** programming language.

## Purpose

The goal of this project is to practice and explore Kotlin concepts through hands-on exercises, covering topics such as:

- Basic syntax and data types
- Control flow (conditions, loops)
- Functions and lambdas
- Object-Oriented Programming (classes, interfaces, inheritance)
- Functional programming
- Collections and extension functions
- Coroutines and asynchronous programming

## Requirements

- [JDK 11+](https://adoptium.net/)
- [Kotlin](https://kotlinlang.org/)
- [Gradle](https://gradle.org/)

## How to Run

Clone the repository and build the project using Gradle:

```bash
git clone https://github.com/your-username/kotlin-laboratory-exercises.git
cd kotlin-laboratory-exercises
./gradlew build
```
